* Quick Start

** Installation

*** From source (recommended)

#+begin_src bash
git clone https://github.com/HaoZeke/rsx-rs.git
cd rsx-rs
cargo build --release
# Binary at target/release/rsx
#+end_src

*** From GitHub releases

Pre-built binaries are available for Linux (x86_64, aarch64), macOS
(x86_64, arm64), and Windows from the
[[https://github.com/HaoZeke/rsx-rs/releases][Releases page]].

*** Via pixi

#+begin_src bash
cd rsx-rs
pixi run build
#+end_src

** Example workflow

Given demultiplexed RAD-seq reads in =reads/= and a population map:

#+begin_example
# popmap.tsv
ind1    M
ind2    M
ind3    F
ind4    F
#+end_example

*** Step 1: Build markers table

#+begin_src bash
rsx process -i reads/ -o markers.tsv -T 4 -d 5
#+end_src

*** Step 2: Check marker frequencies

#+begin_src bash
rsx freq -t markers.tsv -o freq.tsv -d 5
#+end_src

*** Step 3: Compute sex-bias distribution

#+begin_src bash
rsx distrib -t markers.tsv -p popmap.tsv -o distrib.tsv -d 5 -G M,F
#+end_src

*** Step 4: Extract significant markers

#+begin_src bash
rsx signif -t markers.tsv -p popmap.tsv -o signif.tsv -d 5 -G M,F
#+end_src

*** Step 5: Map to reference genome

#+begin_src bash
rsx map -t markers.tsv -p popmap.tsv -g genome.fa -o aligned.tsv -d 5 -G M,F
#+end_src

** Output format

All outputs are tab-separated with an optional =#source:= comment line.
The format is identical to the original C++ RADSex tool, so existing
R scripts (e.g. ChromSex) work without modification.
