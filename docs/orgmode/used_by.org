Publications, theses, and reproducibility packages that use ~rsx~.

* Publications

- *rsx: a high-performance streaming toolkit for RAD-seq sex determination*
  (BMC Bioinformatics). The reference publication: it introduces ~rsx~,
  benchmarks it against C++ RADSex v1.2.0 on four literature RAD-seq panels
  (41.9 billion bases), and reports the Bayesian marker-evidence layer.

* Reproducibility Packages

- [[https://doi.org/10.5281/zenodo.20531539][rsx BMC reproducibility archive (Zenodo)]] :: A pinned pixi + Snakemake
  workflow that regenerates every figure and reported number in the paper. It
  clones ~rsx~ at the tagged release, builds it alongside the C++ RADSex
  v1.2.0 reference, and replays the full literature benchmark matrix.
